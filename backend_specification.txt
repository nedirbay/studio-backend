--------------------------------------------------
3. API ENDPOINTS (LIST)
--------------------------------------------------

Authentication:
- POST /api/auth/login       : Token arkaly ulgam girmek
- GET  /api/auth/me          : Häzirki ulanyjy maglumatlary
- PUT  /api/auth/profile     : Profili täzelemek

Customers:
- GET    /api/customers      : Ähli müşderiler
- POST   /api/customers      : Täze müşderi goşmak
- GET    /api/customers/{id} : Müşderi anyk maglumaty
- PUT    /api/customers/{id} : Müşderini täzelemek
- DELETE /api/customers/{id} : Müşderini öçürmek

Appointments:
- GET    /api/appointments           : Ähli duşuşyklar
- GET    /api/appointments/date/{dt} : Bir günki duşuşyklar
- POST   /api/appointments           : Täze duşuşyk bellemek
- PUT    /api/appointments/{id}      : Ýagdaýyny üýtgetmek

Orders:
- GET    /api/orders                 : Ähli sargytlar
- POST   /api/orders                 : Täze sargyt döretmek (günler bilen)
- GET    /api/orders/{id}            : Sargyt maglumaty (detallary bilen)
- PUT    /api/orders/{id}            : Sargyt täzelemek
- DELETE /api/orders/{id}            : Sarget öçürmek
- GET    /api/orders/staff/{userId}  : Bir işgäriň sargytlary

Equipments:
- GET    /api/equipments             : Ähli enjamlar
- GET    /api/equipments/assigned    : Kimde näme bar (assigned)
- POST   /api/equipments             : Täze enjam goşmak

Expenses & Stats:
- GET    /api/expenses               : Ähli çykdajylar
- POST   /api/expenses               : Täze çykdajy goşmak
- GET    /api/stats/financial        : Umumy girdeji/çykdajy hasabaty

4. DATABASE CONTEXT (EF CORE)
--------------------------------------------------
public class AppDbContext : DbContext
{
    public AppDbContext(DbContextOptions<AppDbContext> options) : base(options) { }

    public DbSet<Customer> Customers => Set<Customer>();
    public DbSet<Appointment> Appointments => Set<Appointment>();
    public DbSet<User> Users => Set<User>();
    public DbSet<Role> Roles => Set<Role>();
    public DbSet<Order> Orders => Set<Order>();
    public DbSet<OrderDay> OrderDays => Set<OrderDay>();
    public DbSet<Expense> Expenses => Set<Expense>();
    public DbSet<Equipment> Equipments => Set<Equipment>();
    public DbSet<GalleryItem> Gallery => Set<GalleryItem>();
}
